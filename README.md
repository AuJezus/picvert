# picvert
### Usage
```
npm i -g
picvert <input directory>
```

or

```
npx picvert <input directory>
```
### Options
 - -h for help.
 - -o for selecting output directory.
 - -f for selecting the format.

Or don't specify any options and the program will ask you.
